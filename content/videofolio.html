Portfolio 2012

Responsive in design én interface.

Wat een mooi Droste-effect; een portfolio-item in een portfolio-item! Bij het maken van dit portfolio heb ik een responsive design gecombineerd met een zelfgeschreven jQuery UI widget.

Bij deze nieuwe versie worden de pagina's en de inhoud op conventionele wijze getoond; goed leesbaar en geoptimaliseerd voor het apparaat waarmee de website wordt bezocht. Een flexibel 'grid' en typografie zorgen dat de layout goed blijft zelfs wanneer de gebruiker op een mobiele telefoon of een belachelijk groot beeldscherm de website bekijkt.

De interface is ook responsive. De manier waarop je door de pagina's navigeert, is afhankelijk van het apparaat. Op telefoon en tablet krijg je daardoor meer het gevoel dat je een native app onder je duim hebt. (todo: dmv swipen)
Bij een flexibele layout zijn de afmetingen leidend maar bij een verandering in de interface is het relevanter om te kijken naar de mogelijke features. Hiervoor gebruik ik Modernizr en check ik bijvoorbeeld op touch events en css3 animations.

Op apparaten met ondersteuning voor touch events kan je door de pagina's heen swipen en worden de volgende en vorige pagina dynamisch ingeladen. Dit is geïnspireerd op bestaande frameworks voor mobiele webapps zoals jQuery Mobile en JQTouch. Een grote uitdaging hierbij was om het swipen niet in conflict te laten komen met het scrollen. Dit ligt nogal dicht bij elkaar en is onder andere moeilijk omdat je vinger een stuk minder nauwkeurige input levert dan een cursor.

Een andere uitdaging was om de scrollpositie van de huidige pagina niet de zichtbaarheid te laten beïnvloeden van de volgende pagina.

De context waarin een website bezocht wordt, is ook iets wat verschilt per apparaat. De start van dit project was dan ook het kort definiëren met welke bedoeling de bezoeker deze website bekijkt.
